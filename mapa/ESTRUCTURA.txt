   └─ Instalable en Android, iOS, Desktop

🚀 INICIO RÁPIDO

# Probar localmente
python servidor.py
# o
python -m http.server 8000

# Abrir navegador
http://localhost:8000

# Subir a hosting
1. Comprimir carpeta mapa/
2. Subir a hosting (public_html/)
3. Activar HTTPS/SSL
4. Acceder a tu dominio
