📋 ¿QUÉ ES ESTE PROYECTO?
═════════════════════════════════════════════════════════════════════════════

Una Progressive Web App (PWA) completa y funcional del mapa interactivo del
municipio de Puerres, Nariño, Colombia.

✨ CARACTERÍSTICAS:
  • 🌐 Funciona como aplicación nativa (Android, iOS, Desktop)
  • 📱 Se puede instalar en cualquier dispositivo
  • ✈️  Funciona SIN INTERNET (después de primera visita)
  • 🌍 Bilingüe: Español e Inglés
  • 🗺️  Mapa interactivo con Leaflet
  • 📍 6 puntos de interés + 12 calles mapeadas
  • 📱 Diseño responsive (móvil, tablet, desktop)


🚀 INICIO RÁPIDO (3 PASOS)
═════════════════════════════════════════════════════════════════════════════

PASO 1: PROBAR LOCALMENTE (2 minutos)
─────────────────────────────────────────────────────────────────────────────

  OPCIÓN A - Con script automático:
  
    1. Doble click en: INICIO-WINDOWS.bat
    2. Se abrirá el navegador automáticamente
    3. ¡Listo!
  
  
  OPCIÓN B - Manualmente:
  
    1. Abre PowerShell o CMD en esta carpeta
    2. Ejecuta: python -m http.server 8000
    3. Abre navegador: http://localhost:8000
  
  
  OPCIÓN C - Con Python script:
  
    1. Ejecuta: python servidor.py
    2. Se abrirá automáticamente


PASO 2: SUBIR A TU HOSTING (5 minutos)
─────────────────────────────────────────────────────────────────────────────

  1. Comprime toda la carpeta "mapa"
     → Click derecho → "Comprimir en ZIP"
  
  2. Accede a tu hosting (cPanel/DirectAdmin)
     → Administrador de archivos
  
  3. Ve a la carpeta "public_html"
     → Sube el archivo ZIP
  
  4. Descomprime el archivo en el servidor
  
  5. Activa HTTPS/SSL (Let's Encrypt - gratis)
  
  6. Accede a: https://tudominio.com
  
  ¡LISTO! Tu PWA está en línea.


PASO 3: PERSONALIZAR (10 minutos)
─────────────────────────────────────────────────────────────────────────────

  1. Genera iconos PNG reales
     → Lee: GENERAR-ICONOS.md
  
  2. Edita ubicaciones del mapa
     → Archivo: js/map.js
  
  3. Personaliza puntos de interés
     → Archivo: js/map.js
  
  4. Cambia colores si quieres
     → Archivo: css/style.css

⚠️ IMPORTANTE - ANTES DE SUBIR A PRODUCCIÓN
═════════════════════════════════════════════════════════════════════════════

  1. GENERAR ICONOS PNG REALES
     Los iconos actuales son SVG temporales. Para producción REAL:
     
     → Usa: https://www.pwabuilder.com/imageGenerator
     → Sube un logo de 512x512px
     → Descarga todos los tamaños
     → Reemplaza en carpeta "icons/"
     
     Ver guía completa: GENERAR-ICONOS.md
  
  
  2. ACTIVAR HTTPS
     PWA requiere HTTPS obligatoriamente. Sin SSL NO funcionará:
     
     → En tu hosting, busca "SSL/TLS"
     → Activa "Let's Encrypt" (gratis)
     → Espera 5-10 minutos
     
     Si usas Netlify/Vercel → HTTPS automático ✅
  
  
  3. PERSONALIZAR UBICACIONES
     Actualiza las coordenadas GPS reales de Puerres:
     
     → Archivo: js/map.js
     → Busca: const PUERRES_CENTER = [...]
     → Cambia latitud y longitud

📱 CÓMO INSTALAR LA APP (Para Usuarios Finales)
═════════════════════════════════════════════════════════════════════════════

  ANDROID (Chrome):
    1. Abre el sitio web
    2. Toca el menú (⋮)
    3. "Añadir a pantalla de inicio"
    4. ¡Listo! Icono en tu pantalla
  
  
  iOS (Safari - NO Chrome):
    1. Abre en Safari
    2. Toca botón compartir (□↑)
    3. "Añadir a pantalla de inicio"
    4. ¡Listo!
  
  
  DESKTOP (Chrome/Edge):
    1. Busca icono ⊕ en barra de direcciones
    2. Click "Instalar"
    3. Se abre como app independiente

🔧 PERSONALIZACIÓN RÁPIDA
═════════════════════════════════════════════════════════════════════════════

  Cambiar nombre de la app:
    → manifest.json → "name": "Tu Nuevo Nombre"
  
  Cambiar ubicación del mapa:
    → js/map.js → const PUERRES_CENTER = [LAT, LNG]
  
  Añadir punto de interés:
    → js/map.js → array pointsOfInterest
    → Añade nuevo objeto con coords, nombre, icono
  
  Cambiar colores:
    → css/style.css → :root { --primary-color: #... }
  
  Añadir idioma (ej: francés):
    → js/translations.js → añade bloque "fr: {...}"
